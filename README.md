# QuickCode

**QuickCode** 是一个小工具脚本库，旨在提升效率并解决常见问题。

## 安装

1. 克隆仓库并进入目录：

   ```bash
   git clone https://github.com/你的用户名/quickcode.git
   cd quickcode
   ```

2. 安装依赖：

   ```bash
   npm install
   ```

## 使用方法

+ **查询 CPU 温度**  
  指令：`npm run system:cpu-temp`  
  使用效果：输出当前 CPU 温度信息，例如 `当前 CPU 温度: 55°C, 最大 CPU 温度: 65°C`。
